# Snake-Water-Gun-GAME
Author-Mahak Sharma<br>
Snake Water Gun is a simple and fun hand game, similar to Rock Paper Scissors.
<br>
<p>
  <b>How to play?</b><br>
  It is played between the user and the computer.<br>The user will select one of these:
  <ul>
    <li>Snake</li>
    <li>Water</li>
    <li>Gun</li>
  </ul>
  <b><u>RULES:</u></b><br>
  <ul>
    <li>Snake drinks Water</li>
    <li>Gun kills Snake</li>
    <li>Water douses Gun</li>
    <li>Draw if BOth selected the same Options</li>
  </ul>
  <br>
  <b>MULTIPLE ROUNDS WILL BE THERE AS PER YOUR CHOICE</b>
</p>
